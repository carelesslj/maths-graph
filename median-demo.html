中线 EP 的取值范围

本题答案：1 < x < 4
P 是 DF 的中点，所以 EP 是从 E 出发的中线。动画让夹角从接近 0° 到接近 180°，三角形始终不退化，因此端点 1 和 4 都取不到。

动态观察

当前 EP = x 0.00
趋近 1 趋近 4

夹角 ∠DEF = 64°

推导

设 DF = a，DE = 3，EF = 5，EP = x。
中线公式：x² = (2·DE² + 2·EF² - DF²) / 4
x² = (2·3² + 2·5² - a²) / 4 = (68 - a²) / 4
由三角形两边和差关系：2 < a < 8。
a 越大，x 越小，所以 1 < x < 4。

这一类题的通用公式

若三角形中，从两边长为 p、q 的夹角顶点作中线，设中线长为 m，则 |p - q| / 2 < m < (p + q) / 2
DE = 3 EF = 5 EP = x
